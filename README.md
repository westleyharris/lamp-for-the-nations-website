# Lamp for the Nations

A multipage website for Lamp for the Nations.

## Pages

- Home (index.html)
- About
- Academics
- Enroll
- Staff
- Donate

## Project Structure

```
├── index.html          # Home page
├── about.html          # About page
├── academics.html      # Academics page
├── enroll.html         # Enroll page
├── staff.html          # Staff page
├── donate.html         # Donate page
├── css/
│   └── main.css        # Main stylesheet
├── js/
│   └── main.js         # Main JavaScript
└── images/             # Image assets
```

## Setup

This site is hosted on GitHub Pages. To view locally, simply open any HTML file in a web browser.
